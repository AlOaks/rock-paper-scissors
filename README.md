#ROCK PAPER SCISSORS LIZARD SPOCK

## Description

Just a little game created out of the game with the same name on the TV show "The Big Bang Theory".
I hope you enjoy it.

## Rules

    Rock beats Scissors
    Scissors beats Paper
    Paper beats Rock
    Rock beats Lizard
    Lizard beats Spock
    Spock beats Rock
    Lizard beats Paper
    Scissors beats Lizard
